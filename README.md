# Face Mask Detection using Deep Learning

## 📌 Overview

This project is a Deep Learning-based Face Mask Detection system that detects whether a person is wearing a face mask or not in real time using computer vision and transfer learning techniques.

The model is built using TensorFlow, Keras, OpenCV, and MobileNetV2, achieving high validation accuracy for mask classification.

---

## 🚀 Features

* Real-time face mask detection
* Image and video mask detection support
* Deep Learning-based classification
* Transfer Learning using MobileNetV2
* High accuracy prediction
* OpenCV-based face detection

---

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/adeeba5260/face-mask-detection.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd face-mask-detection
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Run Real-Time Video Detection

```bash
python detect_mask_video.py
```

### Run Image Detection

```bash
python detect_mask_image.py
```

### Train the Model

```bash
python train_mask_detector.py
```
